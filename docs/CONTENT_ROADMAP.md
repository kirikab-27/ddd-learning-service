
- [ ] レッスンコンテンツ（Markdown）が作成されている

## コンテンツ作成ガイドライン

### レッスン構成

各レッスンは以下の要素で構成:

| 要素 | 説明 | 所要時間目安 |
|------|------|-------------|
| 概念説明 | 図解を含む解説テキスト | 10-15分 |
| コード例 | TypeScriptによる実装サンプル | 5-10分 |
| まとめ | 重要ポイントの整理 | 2-3分 |

### クイズ構成

| 項目 | 仕様 |
|------|------|
| 問題数 | 5問/レッスン |
| 形式 | 選択式（4択） |
| 解説 | 正答・誤答の理由を記載 |
| 合格基準 | 70%以上（3.5問以上） |

### Markdown形式

```markdown
# レッスンタイトル

## 概要
このレッスンで学ぶこと...

## 本文
### セクション1
説明テキスト...

### コード例
\`\`\`typescript
// 実装例
\`\`\`

## まとめ
- ポイント1
- ポイント2
- ポイント3
```
